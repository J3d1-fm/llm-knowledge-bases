LLM Knowledge Bases Changelog

Version v0.6.1 - 2026-06-15
- Tightened global work DB snippet indexing to avoid reading JSON/YAML/TOML/config-like files and noisy cache/worktree folders.
- Added snippet redaction for API-key-like values, token-like fields, and emails.
- Updated version metadata and technical documentation for the work DB safety patch.

Version v0.6.0 - 2026-06-15
- Added a private global work database generator and CLI.
- Added indexing for local Codex Projects, Codex daily workspaces, Codex memory files, Codex skills, Claude files, local git remotes, Codex session index, and Claude session/task metadata.
- Added optional GitHub, GCloud, and Firebase external inventory refresh.
- Added generated private outputs: db.json, files.jsonl, projects.jsonl, sessions.jsonl, tag-cloud.html, and summary README.
- Added safety rules that skip generated dependency/build folders and avoid reading secret-looking file contents.
- Updated documentation with global work database commands and privacy boundaries.

Version v0.5.0 - 2026-06-15
- Added a local vault CLI for stats, search, show, health, ingest, and raw-source registration.
- Added recursive vault markdown loading so imported corpora can be grouped in subfolders.
- Added vault/inbox as a staging folder for unsorted local source exports.
- Added package scripts for kb, kb:stats, and kb:health.
- Updated README and technical documentation with the real-base import and CLI access workflow.

Version v0.4.0 - 2026-06-15
- Added the first expanded vault content batch with 10 new raw notes.
- Added 10 new compiled wiki articles covering LLM-owned wiki maintenance, ingestion, Obsidian, market mapping, private workspace, retrieval, outputs, evaluation, and finetuning.
- Added 10 matching source records for the new raw notes and updated existing source backlinks.
- Added new health checks for raw source registration and market research gaps.
- Added the First Vault Expansion Plan output.
- Added derived rawCount metadata to the Firestore seed snapshot.
- Added validation requiring every raw file to have a matching source record.
- Updated README and technical documentation for the expanded content batch.

Version v0.3.0 - 2026-06-15
