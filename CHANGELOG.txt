LLM Knowledge Bases Changelog

Version v0.8.1 - 2026-06-16
- Fixed the memory graph zoom-out floor so the full clustered map can fit on screen.
- Renamed the graph fit control to Fit all and added explicit zoom out / zoom in buttons.
- Added generated graph state markers for zoom and screen bounds so local QA can verify that the graph fits inside the viewport.
- Updated workdb graph validation to catch regressions to the old zoom limits.

Version v0.8.0 - 2026-06-16
- Rebuilt the private tag-cloud output as a clustered memory graph instead of a ring-style node map.
- Added explicit theme cluster nodes and cluster membership for tags, projects, raw samples, sessions, repos, and cloud resources.
- Added hover magnification, pan/zoom, cluster focus controls, search focus, and a hidden-by-default inspector drawer so no details window blocks the initial graph.
- Added graph actions that expose analysis commands for selected tags and clusters.
- Added workdb analyze-tag and analyze-cluster commands that generate markdown dossiers under outputs/global-work-kb/analysis/.
- Added a local workdb serve mode so the graph can run tag/cluster analysis through a localhost API and open the generated report.
- Added full-scope analysis manifests for tag/cluster drill-downs, including all matched files, projects, sessions, and related tags.
- Added workdb graph validation to the release validation path.
- Updated docs and version metadata for the clustered memory graph release.

Version v0.7.0 - 2026-06-15
- Replaced the indirect tag cloud with an Obsidian/Hermes-style interactive memory graph centered on an agent memory lens.
- Added graph nodes for projects, raw file samples, recent agent sessions, semantic tags, GitHub repos, GCloud projects, and Firebase projects.
- Added generated raw-registry.jsonl, chronology.jsonl, catalog.md, and provenance.md outputs for a raw/wiki/provenance workflow.
- Added stable hashed IDs for file, project, and session records, metadata-only compile statuses, chronology safety fields, and redacted credential-style git remotes.
- Updated global work DB docs and version metadata for the memory-graph release.

Version v0.6.1 - 2026-06-15
