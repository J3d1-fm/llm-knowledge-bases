# Output Formats Notes

The strongest outputs are easy to reopen, diff, cite, and file back into the vault. Markdown reports, Marp slide decks, CSV tables, matplotlib charts, and graph images fit this model.

The output format should follow the question. A strategic answer may become a report, a narrative may become a deck, a comparison may become a table, and a structure question may become a graph.

Each output should keep source coverage and a decision state: filed into wiki, kept as standalone output, needs review, or discarded.
