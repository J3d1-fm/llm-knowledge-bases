# Example Compiled Wiki Output

A compiled wiki page should not be just a polished summary. It should preserve enough structure for later reasoning.

Minimum useful fields include title, summary, source references, linked concepts, confidence, tags, and body paragraphs. Better pages also include open questions, last reviewed date, claim-level citations, and known contradictions.

The key quality bar is reuse. If the next LLM session can read the page and understand what is known, why it is believed, and what remains uncertain, the page is doing useful work.
